# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-04

### Added
- Initial release of recipe-unit-converter
- Core conversion functionality for volume, weight, and temperature units
- Support for cross-category conversions using ingredient densities
- Enhanced parsing with fraction support (1/2, 1 1/2)
- Natural language number parsing (one, two, half, quarter)
- CLI with multiple output formats (simple, verbose, JSON)
- Comprehensive test suite with 107 tests
- Full type hints with strict mypy compliance
- Pydantic models for data validation
- Bundled data files for units and ingredients
- Documentation with installation and usage examples
